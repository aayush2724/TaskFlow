## Overview

TaskFlow is a To-Do List and Productivity Tracker with modern UI and gamification.
It is built as a monorepo containing a React (Vite) frontend and a Node.js/Express backend communicating with a MongoDB database.

## Data Flow

1. User interacts with a React Component (e.g., creating a task on `TasksPage`).
2. Redux Thunk dispatches an asynchronous action using Axios.
3. Axios sends an HTTP request to the Express backend (e.g., `POST /api/tasks`).
4. Express Router forwards the request to the relevant Controller.
5. Controller validates the data and interacts with MongoDB via Mongoose.
6. Mongoose returns the result, Controller sends JSON response back to the Frontend.
7. Redux slice updates the local state, triggering a re-render of the Component.
