# TaskFlow

TaskFlow is a full-stack productivity tracker built with React, Tailwind CSS, Redux Toolkit, Express, and MongoDB.

## Workspaces

- `backend`: Express API with JWT auth, task management, analytics, and gamification.
- `frontend`: Vite + React application with dashboard, tasks, calendar, pomodoro, and profile pages.

## Quick Start

1. Install dependencies:

   ```bash
   npm run install:all
   ```

2. Copy environment files:

   - `backend/.env.example` -> `backend/.env`
   - `frontend/.env.example` -> `frontend/.env`

3. Start the app:

   ```bash
   npm run dev
   ```

## Default Ports

- Frontend: `5173`
- Backend: `5000`
